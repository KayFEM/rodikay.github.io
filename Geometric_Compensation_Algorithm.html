Geometric Compensation Algorithm
Rodi Kay
January 2023
During the process of Additive Manufacturing (AM), the temperature gradients are substantial and transient, resulting
in irreversible residual stresses and plastic deformation that have a negative effect on the materials strength, fatigue
resistance, ductility, and geometrical tolerance. These variants may not meet the necessary requirements.
A solution to this issue is to modify the initial geometry in order to compensate the nal product, for undesired warpage.
In order to print components, a numerical control language is used for the AM machinery. However, there are several
steps to obatin the control commands. This includes creating a CAD model, converting it to an STL le, and using a
software called a Slicer to generate numerical control commands for the AM machinery, usually a GCode.
The most utilized solution to modify the geometry is to change the coordinates of a mesh. However, this solution is
needlessly complex and ineffective. Therefore, a more effective algorithm that instantly modies the GCode is created.
One example where this algorithm was used on an industrial component, and successfully reducing the warpage nu-
merically, is shown below. First, the numerical simulation is done, in order to replicate the actual manufacturing, by
utilizing a thermal-mechanical model, with a visco-plastic-elasto-plastic framework. Where the displacement eld is
derived from, but the designer can use whatever source for the displacement eld. The following gure illustrates the
distortions from the original scanning strategy (GCode) from an industrial impeller.
However, after utilizing the algorithm, and by only considering the wings of the impeller in the compensation part of
the algorithm. The results suggests that the algorithm is a robust method to compensate for distortions in AM processes,
after reducing the distortions by 40

# Find the index of the closest point in gcodePoints

# Return the closest point and its index in gcodePoints

def change_coord(line:str, letter:str, new_value:float)->str:
old_value = getCoord(line, letter)
cells = line.split(" ")
for idx, motion in enumerate(cells):
if motion[0] == letter:
cells[idx] = motion.replace(old_value, new_value)
return " ".join(cells)
